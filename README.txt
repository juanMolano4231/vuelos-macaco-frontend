Aplicación de venta de tiquetes de avión.

Vistas:
	Register (el usuario se puede registrar)
	Login (el usuario hace login)
		Dashboard del usuario (ve información resumida de su cuenta, su estado de tiquetes, cualquier cosa)
			Browsing de viajes (puede buscar viajes para comprarlos)
				Detalles del viaje para comprar (le muestra los detalles del viaje y le permite comprar un tiquete para el viaje)
			Tiquetes ya comprados (le muestra una lista de sus tiquetes ya comprados y para qué viaje son)
				Detalles del tiquete para información (le muestra los detalles del tiquete y viaje pero no ofrece ninguna acción)
