Aplicación de venta de tiquetes de avión.

Vistas:
	Login (el usuario hace login)
		Dashboard del usuario (ve información resumida de su cuenta, su estado de tiquetes, cualquier cosa)
			Browsing de tiquetes (puede buscar tiquetes para comprarlos)
				Detalles del tiquete para comprar (le muestra los detalles del tiquete y le permite comprarlo)
			Tiquetes ya comprados (le muestra una lista de sus tiquetes ya comprados)
				Detalles del tiquete para información (le muestra los detalles del tiquete pero no ofrece ninguna acción)
